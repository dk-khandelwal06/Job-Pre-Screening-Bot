## 📌 Overview

Job Pre-Screening Bot is a **next-generation AI recruitment system** designed to streamline the hiring process by intelligently analyzing candidate inputs and simulating real-world screening workflows.

With a **premium UI + chatbot interaction**, it mimics how modern companies shortlist candidates efficiently using AI.

> 🎯 Designed to automate hiring workflows and enhance recruitment efficiency.

---

## ✨ Features

* 🤖 AI Chatbot-Based Screening System
* 🎯 Automated Candidate Evaluation Flow
* 🧠 Smart Input Processing (Name, Role, Experience, Skills)
* ⚡ Real-Time Interaction & Response System
* 🎨 Futuristic UI (Glassmorphism + Neon Effects)
* 📊 Candidate Profile Simulation
* 🔍 Intelligent Shortlisting Logic
* 📱 Fully Responsive Design

---

## 🧠 How It Works

The system follows a **step-by-step screening pipeline**:

1. 👤 Collect candidate details
2. 🎯 Identify job role
3. 📊 Evaluate experience level
4. 🧠 Analyze technical skills
5. ✅ Generate screening result

👉 Candidate is added to a simulated **AI shortlist database**

---

## 💡 User Flow

1. User clicks **"Deploy AI Recruiter"** 🚀
2. Chatbot initiates interaction 🤖
3. User provides:

   * Name
   * Role
   * Experience
   * Skills
4. System processes inputs
5. 🎯 Displays final screening result
